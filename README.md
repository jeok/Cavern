# Cavern
My friend and I are set on a journey to build a game during one weekend...
and we failed.

## About
**Cavern** is a small 2D exploration game written in **Python 3**.

To play **Cavern** you need these modules installed:
- pyglet
- pytmx

#### Linux/OS X
`pip install pyglet pytmx`
